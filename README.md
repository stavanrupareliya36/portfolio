# Stavan Rupareliya Portfolio

Static portfolio website for Stavan Rupareliya, focused on embedded AI, computer vision, edge computing, and VLSI/design-verification projects.

## Files

- `index.html` - Single-page portfolio
- `styles.css` - Responsive styling and animations
- `script.js` - Small interactions for counters, navigation, and canvas background
- `assets/Stavan_Rupareliya_CV.pdf` - Downloadable resume

## Deploy

This site is designed to run on GitHub Pages from the repository root.
